
<h2>📈 Descripción general:</h2>
<br>
<b>La regresión logística es un modelo estadístico utilizado para problemas de 'clasificación binaria'.</b> En lugar de predecir valores continuos, estima la 'probabilidad' de que una observación pertenezca a una clase (ej. 0 = 'no', 1 = 'sí').   En esta actividad se establece un modelo de regresión logística en Python enfocado a predecir cuántos empleados abandonan o permanecen en una compañía.
